# Preland OnlyFans Style — FB/IG Ads Redirect

Преленд для увода из рекламы Facebook/Instagram (Stories, Reels, Feed) на внешний оффер.

## 🚀 Деплой на GitHub Pages

1. Создай репозиторий: `preland-onlyfans`
2. Загрузи все файлы
3. Settings → Pages → Source: `main branch` / root
4. Готово: `https://username.github.io/preland-onlyfans/`

## ⚙️ Настройка

Открой `config.js` и замени:

```js
OFFER_URL: "https://your-offer-link.com/landing"  // твоя ссылка
MODEL_NAME: "Sophia_Blue"                          // имя модели
FB_PIXEL_ID: "123456789"                           // ID пикселя FB (опционально)
